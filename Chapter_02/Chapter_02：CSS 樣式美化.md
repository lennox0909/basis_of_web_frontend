# **第二課：CSS 樣式美化 — 網頁的皮囊**
* 展示了如何使用 Flexbox 佈局和 Card UI 設計
* 這一課的重點在於 CSS Box Model (盒模型) 和 Flexbox。


## **1\. 核心概念**

CSS (Cascading Style Sheets) 用於控制網頁的視覺呈現。

## **2\. 選擇器 (Selectors)**

* **標籤選擇器**：直接抓取 HTML 標籤（如 body, h1）。  
* **類別選擇器 (Class)**：抓取自定義的屬性（如 .container）。

## **3\. 盒模型 (Box Model)**

網頁上每個元素都是一個方框：

* **Padding (內距)**：內容與邊框之間的距離。  
* **Border (邊框)**：圍繞內容的線。  
* **Margin (外距)**：元素與元素之間的距離。

## **4\. 佈局基礎**

* display: flex;：現代排版最核心的工具。  
* justify-content：水平對齊。  
* align-items：垂直對齊。
