# **第五課：與後端溝通 (Fetch API) — 數據聯網**

## **1\. 核心概念**

讓網頁具備抓取雲端資料的能力。

## **2\. Async / Await**

網路請求需要時間等待，使用非同步語法確保程式不會在等待時「卡死」：

* async：標記函式為非同步。  
* await：暫停執行，直到資料傳回。

## **3\. Fetch API**

* fetch(URL)：向該 URL 發出請求。  
* .json()：將原始資料包轉譯為 JS 物件。

## **4\. 流程設計**

1. 發出請求 \-\> 2\. 顯示載入中 \-\> 3\. 成功拿到資料 \-\> 4\. 渲染到 HTML \-\> 5\. 錯誤處理 (Error Handling)。
